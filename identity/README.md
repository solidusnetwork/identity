# Solidus Identity

Comprehensive identity creation, management, and verification platform.

- **Domain:** identity.solidus.network
- **Status:** Planned
- **Docs:** `../../docs/protocol.md`, `../../docs/architecture.md`

## Overview

WebID-based identity management with verifiable credentials wallet, privacy controls, and federation services. Users create and control their decentralized identifier (DID), manage credentials, and authenticate across services.

## Key Features

- DID creation and management (`did:solidus`)
- Verifiable credentials wallet
- Selective disclosure controls
- OAuth/OpenID Connect bridge
- Multi-device synchronization
